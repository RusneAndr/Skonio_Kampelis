# Skonio Kampelis

## Projekto aprašymas

„Skonio Kampelis“ – tai paprasta kelių puslapių HTML svetainė, skirta pateikti greitus ir lengvai pagaminamus receptus.

Projektas buvo sukurtas kaip studijų darbas, siekiant praktiškai pritaikyti interneto svetainių kūrimo pagrindus.

## Svetainės funkcijos

Svetainė turi tris pagrindinius puslapius:

### Pagrindinis puslapis (index.html)
- Svetainės pristatymas
- Pagrindinė hero sekcija
- Nuorodos į receptų puslapį

### Receptų puslapis (recipes.html)
- pusryčių receptas
- pietų receptas
- vakarienės receptas
- ingredientų lentelės
- receptų palyginimo lentelė

### Kontaktų puslapis (contact.html)
- Kontaktinė forma
- Laukai: vardas, el. paštas, tema, žinutė
- Submit mygtukas

## Kaip paleisti projektą

1. Atsisiųsti projekto failus.
2. Atidaryti projekto aplanką.
3. Atidaryti failą **index.html** naršyklėje.

Papildomas serveris nereikalingas, nes tai yra statinė svetainė.
